# Maze-Game-Python


## Các file quan trọng
 - menu.py: Mở ra menu giao diện game
 -editor.py: Dùng để thêm map mới
 -main.py: Game chính

## Sử dụng
 - Edit map : 
  + Dùng W,S,A,D để di chuyển lên xuống, trái, phải
  + Ctrl + S để lưu map
- Game:
    +Dùng phím mũi tên để di chuyển nhân vật
